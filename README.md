efrons
======

A C++11 program to enumerate [Efron's Dice](http://mathworld.wolfram.com/EfronsDice.html).

How to build
------------

With [GCC](https://gcc.gnu.org/)
```
$ g++ -std=c++11 -o efrons efrons.cc
```
Then run `./efrons`.
